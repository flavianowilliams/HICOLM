# Installation

To install just run "as a superuser" the script install.sh in the SRC directory,

```
$ sudo ./install.sh
```

Notice: gfortran libraries is necessary to compile the Hicolm program.

# Usage

* Amber Protocol available in Force Field section. :star2:
